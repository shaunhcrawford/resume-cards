## Development

To run this project locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file in the root directory with:

```env
VITE_OPENAI_API_KEY=your_openai_api_key
```
